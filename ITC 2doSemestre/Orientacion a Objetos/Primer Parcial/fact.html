Programa: Factorial

Descripción: Programa que tiene una función para calcular el factorial de un número
Autor: Juan Perez
Fecha de elaboración 28 de Enero de 2017


Función: factorial

Descripción: Calcula el factorial de un número
Parametro: iNum valor para el que se va a calcular el factorial. Debe ser entero positivo.
Valor de retorno: factorial del valor iNum


Función: validaEntero

Descripción: Función que pide al usuario un valor y valida que sea entre 1 y 10
Valor de retorno: regresa el valor entero que tecleó el usuario (entre 1 y 10)


Función: main

Descripción: Pide el valor al usuario, calcula el factorial y lo muestra
